# XML Lexer Example

A lexical analyzer for XML implemented using JavaCC (.jj) grammar.

## Features
- Tokenizes XML documents
- Handles tags, attributes, and content
- Error reporting for malformed XML

## Requirements
- JavaCC (Java Compiler Compiler)
- Java JDK 8+
